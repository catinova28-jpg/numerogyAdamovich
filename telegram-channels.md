## Настройка в Продамус

Для каждого курса/челленджа в Продамус нужно:
1. Зайти в настройки продукта
2. Настроить авто-письмо после оплаты
3. В шаблон письма вставить текст + ссылку на ТГ-канал

Пример текста письма:
```
Поздравляем с покупкой!

Вы приобрели: [Название курса]

Для получения доступа перейдите по ссылке и присоединитесь к закрытому Telegram-каналу:
[ССЫЛКА НА ТГ-КАНАЛ]

Если возникнут вопросы — напишите нашему ассистенту.

С уважением,
Команда Сергея Адамовича
```
