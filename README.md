# Сергей Адамович — Нумерология для предпринимателей

Лендинг-страница с курсами, разборами и записью на диагностику.

## Стек
- HTML / CSS / JavaScript (один файл, без фреймворков)
- Шрифты: Google Fonts (Playfair Display + Inter)
- Оплата: Продамус (внешние ссылки)
- Доступ к курсам: закрытые Telegram-каналы

## Деплой

### Таймвеб (VPS)
1. Загрузить файлы на сервер через SFTP
2. Настроить Nginx для отдачи статики
3. Привязать домен

### Файлы для деплоя
- `index.html` — основная страница
- `sergey.png` — фото автора
